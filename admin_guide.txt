Руководство по использованию админки сайта "Школа фехтования"
============================================================

1. Вход в админ-панель
----------------------
- Для входа в админку используйте свой логин и пароль администратора.
- После входа вы увидите навигацию по разделам: Заявки, Тренеры, Галерея, Пользователи, Страницы, Главная.

2. Раздел "Заявки"
-------------------
- Здесь отображаются все заявки пользователей на тренировки.
- Для удаления заявки нажмите кнопку "Удалить" рядом с нужной заявкой.
- После удаления список обновится автоматически.

3. Раздел "Тренеры"
--------------------
- Для добавления тренера заполните форму: имя, описание, фото (обязательно).
- После заполнения нажмите "Добавить тренера".
- Новый тренер появится в списке ниже.
- Для удаления тренера нажмите "Удалить" рядом с нужным тренером.
- После удаления список обновится.

4. Раздел "Галерея"
--------------------
- Для добавления изображения:
  1. Выберите тип "Изображение".
  2. Заполните название, описание (по желанию), выберите файл.
  3. Нажмите "Добавить".
- Для добавления видео:
  1. Выберите тип "Видео".
  2. Заполните название, описание (по желанию), вставьте ссылку на видео (YouTube, Rutube и др.).
  3. Нажмите "Добавить".
- Для удаления элемента галереи нажмите "Удалить" рядом с нужным элементом.

5. Раздел "Пользователи"
-------------------------
- Здесь отображаются все пользователи сайта.
- Для смены роли пользователя (админ/пользователь) нажмите соответствующую кнопку.
- После смены роли список обновится.

6. Раздел "Страницы" (CMS)
---------------------------
- Для создания новой страницы:
  1. В выпадающем списке "Slug страницы" выберите "Другое (ввести вручную)".
  2. Введите уникальный slug (например: faq, news и т.д.).
  3. Заполните заголовок и контент страницы.
  4. Нажмите "Сохранить страницу".
- Для редактирования существующей страницы:
  1. Нажмите "Редактировать" напротив нужной страницы.
  2. Внесите изменения и нажмите "Сохранить страницу".
- Для удаления страницы нажмите "Удалить" напротив нужной страницы.
- **Важно:**
  - Если вы хотите создать новую страницу, сначала сбросьте форму кнопкой "Очистить".
  - После выбора "Другое (ввести вручную)" обязательно введите slug вручную.

7. Раздел "Главная" (Блоки главной страницы)
---------------------------------------------
- Для создания нового блока:
  1. Сначала нажмите кнопку "Сбросить" (очистить форму), чтобы избежать случайного редактирования существующего блока.
  2. Заполните все поля: тип блока, заголовок, содержимое, позиция, видимость.
  3. Нажмите "Сохранить блок".
  4. После сохранения блок появится в списке ниже.
- Для редактирования блока:
  1. Нажмите "Редактировать" напротив нужного блока.
  2. Внесите изменения и снова нажмите "Сохранить блок".
- Для удаления блока нажмите "Удалить" напротив нужного блока.
- Для загрузки изображений в блок:
  1. Сначала создайте блок и сохраните его.
  2. Затем нажмите "Редактировать" напротив этого блока.
  3. После этого используйте форму загрузки изображений (выберите файл, введите описание, нажмите "Загрузить изображение").
  4. После загрузки изображения появятся в списке ниже.
  5. Для удаления изображения используйте кнопку "Удалить" рядом с нужным изображением.
- **Важно:**
  - Всегда сначала создавайте или выбирайте блок, только потом загружайте изображения.
  - Если вы хотите создать новый блок, обязательно сбросьте форму перед этим.
  - После сохранения блока для загрузки изображений обязательно нажмите "Редактировать".

8. Раздел "Блоки страницы" (CMS)
----------------------------------
- Для управления блоками обычных страниц (например, "О нас", "Контакты" и др.):
  1. В разделе "Страницы" нажмите "Управление блоками" напротив нужной страницы.
  2. Для создания нового блока сначала нажмите "Сбросить".
  3. Заполните все поля и нажмите "Сохранить блок".
  4. Для редактирования блока нажмите "Редактировать" напротив нужного блока.
  5. Для удаления блока нажмите "Удалить".
  6. Для загрузки изображений в блок: выберите или создайте блок, затем используйте форму загрузки изображений.
- **Важно:**
  - Всегда сначала создавайте или выбирайте блок, только потом загружайте изображения.
  - После сохранения блока для загрузки изображений обязательно нажмите "Редактировать".

9. Общие рекомендации
---------------------
- После любого действия (создание, редактирование, удаление) всегда проверяйте появление уведомления в верхней части экрана.
- Если что-то не работает или появляется ошибка — обратите внимание на текст уведомления.
- Для предотвращения случайных изменений всегда используйте кнопки "Сбросить" перед созданием новых блоков или страниц.
- Не загружайте изображения до тех пор, пока не создан и не выбран нужный блок.
- Для корректной работы админки используйте современные браузеры (Chrome, Firefox, Edge).

10. Вопросы и поддержка
------------------------
- Если у вас возникли вопросы по работе админки или что-то не работает, обратитесь к разработчику сайта.
